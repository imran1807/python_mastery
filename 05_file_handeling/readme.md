# 📚 Student Record Manager

A simple command-line Student Record Management System built using Python.

This project demonstrates the fundamentals of Python programming by implementing CRUD (Create, Read, Update, Delete) operations using text files instead of a database.

---

## 🚀 Features

- ➕ Add a new student
- 📋 View all students
- 🔍 Search for a student
- ✏️ Update an existing student
- ❌ Delete a student
- 💾 Persistent storage using a text file (`students.txt`)
- 📂 Uses `pathlib` for platform-independent file handling
